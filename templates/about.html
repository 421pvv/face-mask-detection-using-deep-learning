About This Project

Learn about the technology and implementation details of this face mask detection system

Project Overview

This face mask detection system uses deep learning to identify whether people are wearing face masks properly, improperly, or not at all. It can process both live video streams and uploaded images.

Technical Details

The system is built using the following technologies:

  • Python 3.9 - Core programming language
  • TensorFlow 2.12 - Deep learning framework with Metal acceleration for M1/M2 Macs
  • MobileNetV2 - Convolutional neural network architecture
  • OpenCV - Computer vision library for face detection and image processing
  • Flask - Web framework for the application
  • Bootstrap 5 - UI framework

Model Architecture

The mask detection model is based on the MobileNetV2 architecture with transfer learning. The model was pre-trained on ImageNet and then fine-tuned on our face mask dataset. This approach provides an excellent balance between accuracy and performance, making it suitable for real-time applications.

Usage Instructions

Live Detection

  1. Navigate to the "Live Detection" page
  2. Allow camera access when prompted
  3. Position yourself in front of the camera
  4. The system will automatically detect and classify your mask wearing status

Image Upload

  1. On the home page, use the "Image Upload" card
  2. Click "Choose File" to select an image from your device
  3. Click "Upload & Detect" to process the image
  4. View the results displayed below

System Requirements

For optimal performance, this application is recommended to run on:

  • macOS with Apple Silicon (M1/M2)
  • Python 3.9 with TensorFlow Metal acceleration
  • At least 4GB RAM
  • Webcam for live detection

The application can also run on other systems but may require adjustments to the TensorFlow dependencies.
